 On this repo are described the steps followed to install Ubuntu 22.04.5  
 OS and Abaqus 6.14-5 on a Laptop with a Pentium Gold 6405U processor.  
 
 1.- A live Unbuntu 22.04.1 session was run on the lap with a new and empty  
     without any OS installed.  
     
 2.- From the live session the Unbuntu 22.04.5 was downloaded and a bootable  
     USB was created.
     
 3.- The live session was terminated and the Unbuntu 22.04.5 OS was installed  
     from the bootable USB.  
     
  [ubuntu_22.04.5_install_on_laptop](/ubuntu_22.04.5_install_on_laptop)
     
 4.- Thereafter, we proceeded to install Abaqus 6.14-5.  
     
   [abaqus_6.145_install_on_ubuntu](/abaqus_6.145_install_on_ubuntu_22.04)
 
 5.- The laptop specs were these systems were installed were:  
 
     [link_3](./lap_specs/Intel(R)Pentium(R)CPU6405U@2.40GHz_specs)
     
 6.- The laptop specs were collected with an .sh script.  
 
     [link_4](/lap_specs/hp_pentium_gold_lap_spec_full)
